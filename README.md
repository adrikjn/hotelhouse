HotelHouse
Description
hotelhouse est un site web créé avec PHP Symfony permettant de réserver des chambres d'hôtel. Le site offre différentes fonctionnalités telles que la réservation de chambres, la gestion des comptes administrateurs via Easy Admin, la possibilité de contacter l'hôtel, de laisser des avis, et de s'abonner à une newsletter pour recevoir des informations sur les offres spéciales. L'envoi d'e-mails est géré par Mailer SendinBlue. Le site est conçu pour une utilisation uniquement sur ordinateur de bureau. Il génère automatiquement les photos des carrousels et les ordres avec le backend. De plus, il propose une API pour afficher les actualités du monde.

Installation
Clonez ce dépôt sur votre machine locale.
Assurez-vous d'avoir PHP et Symfony installés sur votre système.
Installez les dépendances en exécutant la commande composer install.
Configurez votre base de données dans le fichier .env.
Exécutez les migrations pour créer les tables de la base de données en utilisant la commande php bin/console doctrine:migrations:migrate.
Lancez le serveur Symfony en exécutant la commande symfony server:start.
Utilisation
Accédez au site via votre navigateur web en vous rendant à l'adresse fournie par Symfony après le démarrage du serveur.
Explorez les différentes fonctionnalités telles que la réservation de chambres, la gestion des comptes administrateurs, la fonctionnalité de contact, les avis, la newsletter, et l'API d'actualités.
